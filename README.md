# 概要 About

Unity のオブジェクトにつけられるメモ帳コンポーネントです。 \
A stickynote component for taking a note in Unity. \
VRChat のアバターやワールドのプロジェクトに使用しても、そのままアップロードすることができます。 \
You can include this to any VRChat projects, Avatars and Worlds; this won't block uploading them!  \
アバターに使用したアセットの URL を列挙して後から配布元にアクセスしやすくしたり、ワールドの動画プレイヤーで参照したプレイリストの URL を保存して更新しやすくしたり、使い方は無限大! \
Can be used ANY purposes, keeping track the asset list (url), helping remember where video playlists come from, and so on! \
\
VRChat 以外の Unity プロジェクトにも使えるよ \
Ycan use this outside of VRC Unity projects! \

## 使い方; usage

0. UnityPackage をインポートする
import the package.
1. メモを追加したい GameObject を選択する \
select gameobject that need to take note
2. インスペクタの "Add Component" を押し "VRC Safe Sticky Note" を追加する  \
Add component > VRC Safe Sticky Node
3. テキスト入力欄に自由にメモする    \
Write down anything as you want!
